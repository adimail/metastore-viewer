Welcome to Metastore Viewer

Metastore Viewer is a data lake management interface designed for interaction with object stores. It provides tools for exploring storage buckets, querying data, and comparing metadata efficiently.

Features

  • Bucket Explorer: Browse and manage your storage buckets.
  • Query Editor: Run SQL queries on your data lakes.
  • Metadata Comparison: Compare metadata of different datasets.
  • Settings: Customize your preferences and configurations.

How to Use

  1. Login to your account or register if you haven't.
  2. Use the Bucket Explorer to browse available storage buckets.
  3. Write and execute SQL queries in the Query Editor.
  4. Compare metadata between datasets in the Metadata Comparison section.
  5. Visit Settings to manage configurations.
